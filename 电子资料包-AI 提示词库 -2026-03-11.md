## 一、写作创作类

### 1.1 文章写作

```
【万能文章生成器】
你是一位专业的内容创作者。请根据以下要求帮我写一篇文章：
- 主题：[填写主题]
- 目标读者：[填写读者群体]
- 文章类型：[干货/故事/观点/评测]
- 字数要求：[填写字数]
- 语气风格：[专业/轻松/幽默/严肃]
- 核心要点：[列出 3-5 个要点]

请确保文章结构清晰，有吸引人的开头，充实的内容，和有力的结尾。
```

```
【爆款标题生成器】
请为以下主题生成 10 个爆款标题：
主题：[填写主题]

要求：
- 使用数字、疑问、对比等技巧
- 制造好奇心和紧迫感
- 符合新媒体传播规律
- 每个标题不超过 25 字
```

```
【文章润色优化】
请帮我优化以下文章：
[粘贴原文]

优化方向：
- 提升逻辑连贯性
- 增强语言感染力
- 优化段落结构
- 修正语法错误
- 保持原有风格
```

```
请帮我创作一部 [类型] 小说的大纲：
- 主题：[填写主题]

- 核心冲突：[主要矛盾]

请提供完整的故事大纲，包括起承转合和关键情节点。
```

```
【角色设定生成】
请帮我设计一个 [类型] 角色：
- 基本信息：[年龄/性别/职业]
- 性格特点：[3-5 个关键词]

- 背景故事：[成长经历]

```

### 1.3 文案写作

```
【产品描述文案】
请为以下产品撰写销售文案：
- 产品名称：[填写名称]
- 核心功能：[列出 3-5 个功能]
- 目标用户：[填写用户群体]
- 价格区间：[填写价格]
- 竞品对比：[优势在哪里]

要求：突出卖点，制造购买冲动，包含行动号召。
```

```
【朋友圈文案生成】
请为以下场景生成 5 条朋友圈文案：
场景：[填写场景，如新品上市/活动促销/日常分享]

要求：
- 简短有力，不超过 100 字

- 适合配图发布
```

## 三、学习辅导类

### 3.1 知识讲解

```
【概念讲解】
请用通俗易懂的方式解释以下概念：
概念：[填写概念]
目标读者：[初学者/有一定基础/专业人士]

要求：
- 用生活中的例子类比
- 避免过度使用专业术语
- 包含核心要点总结
- 提供进一步学习的建议
```

```
【学习路径规划】
请帮我规划 [领域] 的学习路径：
- 当前水平：[零基础/入门/进阶]
- 学习目标：[掌握技能/通过考试/找到工作]
- 可用时间：[每天/每周可投入时间]
- 学习偏好：[视频/书籍/实践]

请提供详细的学习路线，包括：
- 阶段划分
- 每个阶段的学习内容
- 推荐资源
- 预计时间
```

```
- 指出容易出错的地方
```

## 💡 提示词使用技巧

### 1. 结构化输入
- 使用清晰的格式列出要求
- 提供足够的背景信息
- 明确期望的输出形式

### 2. 迭代优化
- 第一次输出不满意时，说明具体问题
- 逐步细化要求
- 保存有效的提示词模板

### 3. 角色设定
- 给 AI 设定专业角色
- 说明目标受众
- 定义语气风格

### 4. 约束条件
- 明确字数限制
- 指定格式要求
- 列出必须包含/避免的内容
