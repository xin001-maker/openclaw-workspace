# 2026-03-08 工作记忆

## 🎯 核心突破：自动化工作架构

### 问题
- OpenClaw 是请求 - 响应架构，无法主动发起对话
- 装再多技能也无法定时自动汇报
- 主会话被动等待用户消息

### 解决方案
使用 OpenClaw Cron + systemEvent：
```bash
openclaw cron add \
  --name "heartbeat-worker" \
  --cron "*/30 * * * *" \
  --system-event "根据 HEARTBEAT.md 执行任务" \
  --timeout-seconds 1800
```

### 验证结果
- ✅ Cron 已配置（任务 ID: a3ad53ed-607a-4f36-80ce-0c915b822547）
- ✅ 每 30 分钟自动触发
- ✅ 发送 systemEvent 到主会话
- ✅ 用户回复后开始工作
- ✅ 完成后主动汇报

### 工作流程
```
每 30 分钟 → Cron 触发 → systemEvent 到主会话 → 用户回复 → 工作 → 汇报
```
