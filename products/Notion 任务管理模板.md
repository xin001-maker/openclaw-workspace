# 📋 个人任务管理系统 - Notion 模板

## 模板说明

这是一个简单高效的个人任务管理系统，适合自由职业者、学生、职场人士使用。

**核心功能：**
- ✅ 任务管理（待办/进行中/已完成）
- 📅 日历视图
- 🎯 目标追踪
- 📊 进度统计
- 🏷️ 标签分类

---

## 数据库结构

### 1. 任务数据库 (Tasks)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 任务名称 | Title | 任务标题 |
| 状态 | Select | 待办 / 进行中 / 已完成 / 已取消 |
| 优先级 | Select | 🔴 高 / 🟡 中 / 🟢 低 |
| 截止日期 | Date | 任务截止时间 |
| 预估时间 | Number | 预计需要的小时数 |
| 实际时间 | Number | 实际花费的小时数 |
| 标签 | Multi-select | 工作 / 学习 / 生活 / 健康 / 财务 |
| 项目 | Relation | 关联到项目数据库 |
| 备注 | Text | 任务详细说明 |

### 2. 项目数据库 (Projects)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 项目名称 | Title | 项目标题 |
| 状态 | Select | 规划中 / 进行中 / 已完成 / 暂停 |
| 开始日期 | Date | 项目开始时间 |
| 目标日期 | Date | 预期完成时间 |
| 完成度 | Number | 0-100% |
| 相关任务 | Relation | 关联到任务数据库 |

### 3. 习惯追踪数据库 (Habits)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 习惯名称 | Title | 如"早起"、"运动"、"阅读" |
| 频率 | Select | 每天 / 每周 X 次 / 每月 |
| 连续天数 | Number | 当前连续完成天数 |
| 总完成次数 | Number | 历史总完成次数 |
| 开始日期 | Date | 习惯养成开始日期 |

## 视图配置

### 任务视图

1. **今日待办** - 筛选：截止日期 = 今天 AND 状态 ≠ 已完成
2. **本周任务** - 筛选：截止日期 ≤ 本周五 AND 状态 ≠ 已完成
3. **按优先级** - 分组：优先级
4. **按标签** - 分组：标签
5. **日历视图** - 按截止日期显示

### 项目视图

1. **进行中项目** - 筛选：状态 = 进行中
2. **项目看板** - 看板视图，按状态分组
3. **项目时间线** - 时间线视图

### 习惯视图

1. **每日打卡** - 表格视图
2. **习惯统计** - 按连续天数排序

## 使用建议

### 每日流程

**早晨（5 分钟）：**
1. 查看"今日待办"视图
2. 确定今天最重要的 3 件事
3. 估算每项任务所需时间

**晚上（5 分钟）：**
1. 更新完成任务的状态
2. 记录实际花费时间
3. 规划明天的任务

### 每周回顾（周日，15 分钟）

1. 检查本周完成情况
2. 更新项目进度
3. 规划下周重点任务
4. 回顾习惯养成情况
