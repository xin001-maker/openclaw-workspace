## 📖 什么是 OpenClaw？

**OpenClaw** = 你的 24 小时个人 AI 助理

它能帮你：
- 📧 自动回复消息
- 📅 管理日程提醒
- 📁 处理文件和数据
- 🌐 上网搜索信息
- 💻 执行电脑操作
- 🔔 主动汇报工作

**简单说**：它是一个能真正帮你干活的 AI，不只是聊天！

## 🛠️ 常见问题

### Q1: AI 不回复怎么办？

**检查**：
1. 系统是否正常运行：`openclaw status`
2. 网络连接是否正常
3. 配置是否正确

**解决**：
```bash
# 重启服务
openclaw gateway restart
```

---

### Q2: 如何更换模型？

**方法**：
```bash
# 查看可用模型
openclaw config get models

# 修改模型配置
openclaw config set models.primary <模型名称>
```

---

### Q3: 如何备份配置？

**方法**：
```bash
# 复制配置文件
cp ~/.openclaw/openclaw.json ~/backup/openclaw.json.bak
```
