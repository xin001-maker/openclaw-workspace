## 📋 执行步骤

### 步骤 1：登录实现网（5 分钟）

1. 访问：https://shixian.com/jobs
2. 点击右上角"登录"
3. 选择登录方式（推荐 GitHub，已有账号）
4. 完成登录

### 步骤 2：完善个人资料（10 分钟）

**必填信息：**
- 昵称
- 技术标签（Python/Node.js/全栈等）
- 工作年限
- 所在城市
- 可工作时间

**个人简介模板：**
```
资深全栈开发者，5 年互联网经验，专注于 Python 开发、数据处理、爬虫和自动化脚本。
熟悉 Web 开发（Flask/Django/Node.js）、数据库设计、API 集成。
工作认真负责，沟通顺畅，2 小时内响应需求，提供 7 天免费维护。
```

### 步骤 3：筛选项目（5 分钟）

**优先投标类型：**
- ✅ Python 开发
- ✅ 数据处理/爬虫
- ✅ Web 开发（Flask/Django/Node.js）
- ✅ 自动化脚本
- ✅ AI 应用集成

**跳过类型：**
- ❌ 硬件/嵌入式
- ❌ 纯设计类
- ❌ 需要特殊资质

### 步骤 4：开始投标（每个 3-5 分钟）

**投标话术模板：**

```
您好！我对这个项目很感兴趣。

【我的优势】
- 5 年 Python 开发经验，类似项目交付 20+
- 熟悉 [项目相关技术]
- 2 小时响应，7 天维护

【项目理解】
[简要描述对项目需求的理解]

【实施方案】
1. 需求确认（1 天）
2. 开发实现（2-3 天）
3. 测试交付（1 天）

【报价】
根据需求复杂度，预算范围：XXX-XXX 元

期待进一步沟通！
```

## 🚀 快速开单技巧

1. **响应速度** - 新项目发布后 1 小时内投标
2. **针对性** - 针对项目需求定制话术，不要复制粘贴
3. **合理报价** - 不要过低（显得不专业），不要过高（吓跑客户）
4. **展示案例** - 提及类似项目经验
5. **承诺明确** - 响应时间、交付时间、维护期
