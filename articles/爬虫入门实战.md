## 第一部分：法律边界（必读！）

### ⚠️ 爬虫的 3 条红线

**1. 不要爬取个人隐私数据**
- ❌ 用户信息、手机号、身份证号
- ✅ 公开的商品信息、新闻文章

**2. 不要绕过付费墙**
- ❌ 付费内容、会员专享
- ✅ 免费公开的内容

**3. 不要搞垮对方服务器**
- ❌ 高频请求、DDoS 攻击
- ✅ 控制频率、礼貌爬取

### ✅ 安全爬取原则

```python
# 加上 User-Agent，表明身份

# https://网站域名/robots.txt
```

## 第三部分：第一个爬虫（30 分钟）

### 目标：抓取豆瓣电影 Top250

**网址：** https://movie.douban.com/top250

**要抓的数据：**
- 电影名称
- 评分
- 评价人数
- 简介

### 步骤 1：分析网页结构

按 F12 打开开发者工具，找到电影信息的 HTML 结构：

```html
<div class="item">
  <div class="info">
    <div class="hd">
      <span class="title">肖申克的救赎</span>
    </div>
    <div class="bd">
      <p class="star">
        <span class="rating45"></span>
        <span class="rating_num">9.7</span>
      </p>
      <p class="quote">
        <span class="inq">希望让人自由</span>
      </p>
    </div>
  </div>
</div>
```

### 步骤 2：写爬虫代码

```python
#!/usr/bin/env python3
import requests
from bs4 import BeautifulSoup
import time
import csv

def crawl_douban_top250():
    """抓取豆瓣电影 Top250"""
    
    base_url = "https://movie.douban.com/top250"
    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
    }
    
    movies = []
    
    # 共 10 页，每页 25 部
    for page in range(10):
        url = f"{base_url}?start={page * 25}"
        print(f"正在爬取第 {page + 1} 页...")
        
        response = requests.get(url, headers=headers)
        soup = BeautifulSoup(response.text, 'html.parser')
        
        items = soup.find_all('div', class_='item')
        
        for item in items:
            try:
                title = item.find('span', class_='title').text
                rating = item.find('span', class_='rating_num').text
                quote_elem = item.find('span', class_='inq')
                quote = quote_elem.text if quote_elem else "无"
                
                movies.append({
                    '标题': title,
                    '评分': rating,
                    '简介': quote
                })
            except:
                continue
        
        time.sleep(1)  # 礼貌爬取，间隔 1 秒
    
    # 保存到 CSV
    save_to_csv(movies, 'douban_top250.csv')
    print(f"完成！共抓取 {len(movies)} 部电影")

def save_to_csv(data, filename):
    """保存数据到 CSV"""
    with open(filename, 'w', newline='', encoding='utf-8-sig') as f:
        writer = csv.DictWriter(f, fieldnames=data[0].keys())
        writer.writeheader()
        writer.writerows(data)

if __name__ == "__main__":
    crawl_douban_top250()
```

## 第四部分：进阶技巧（1 小时）

### 技巧 1：处理动态网页

有些网站用 JavaScript 加载数据，直接 requests 拿不到。

**解决方案：用 Selenium**

```bash
pip install selenium webdriver-manager
```

```python
from selenium import webdriver
from selenium.webdriver.chrome.service import Service
from webdriver_manager.chrome import ChromeDriverManager

def crawl_dynamic_site():
    # 自动下载 ChromeDriver
    driver = webdriver.Chrome(service=Service(ChromeDriverManager().install()))
    
    driver.get("https://example.com")
    
    # 等待页面加载
    driver.implicitly_wait(10)
    
    # 获取渲染后的 HTML
    html = driver.page_source
    
    # 用 BeautifulSoup 解析
    soup = BeautifulSoup(html, 'html.parser')
    
    driver.quit()
```

### 技巧 2：处理反爬机制

**常见反爬手段：**

1. **User-Agent 检测**
   ```python
   # 解决方案：随机 User-Agent
   import random
   user_agents = [
       'Mozilla/5.0 (Windows NT 10.0; Win64; x64)...',
       'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)...',
       'Mozilla/5.0 (X11; Linux x86_64)...'
   ]
   headers = {'User-Agent': random.choice(user_agents)}
   ```

2. **IP 封禁**
   ```python
   # 解决方案：使用代理 IP
   proxies = {
       'http': 'http://proxy_ip:port',
       'https': 'https://proxy_ip:port'
   }
   response = requests.get(url, proxies=proxies)
   ```

3. **登录验证**
   ```python
   # 解决方案：携带 Cookies
   cookies = {
       'session_id': 'your_session_id'
   }
   response = requests.get(url, cookies=cookies)
   ```

### 技巧 3：数据清洗

抓到的数据往往需要清洗：

```python
import re

def clean_data(text):
    # 去除空格
    text = text.strip()
    
    # 去除特殊字符
    text = re.sub(r'[^\w\s\u4e00-\u9fff]', '', text)
    
    # 统一格式
    text = text.replace('\n', ' ')
    
    return text
```

## 第五部分：实战项目（1 小时）

### 项目：电商价格监控

**需求：**
- 监控某商品价格变化
- 降价时自动通知
- 记录历史价格

### 完整代码

```python
#!/usr/bin/env python3
import requests
from bs4 import BeautifulSoup
import json
import time
from datetime import datetime

class PriceMonitor:
    def __init__(self):
        self.headers = {
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64)'
        }
        self.products = []
        self.load_products()
    
    def load_products(self):
        """加载要监控的商品"""
        try:
            with open('products.json', 'r', encoding='utf-8') as f:
                self.products = json.load(f)
        except:
            self.products = []
    
    def add_product(self, name, url, target_price):
        """添加监控商品"""
        self.products.append({
            'name': name,
            'url': url,
            'target_price': target_price
        })
        self.save_products()
    
    def save_products(self):
        """保存商品列表"""
        with open('products.json', 'w', encoding='utf-8') as f:
            json.dump(self.products, f, ensure_ascii=False, indent=2)
    
    def check_price(self, product):
        """检查商品价格"""
        try:
            response = requests.get(product['url'], headers=self.headers)
            soup = BeautifulSoup(response.text, 'html.parser')
            
            # 根据实际网站结构调整选择器
            price_elem = soup.find('span', class_='price')
            current_price = float(price_elem.text.replace('¥', ''))
            
            # 记录价格
            self.record_price(product['name'], current_price)
            
            # 检查是否降价
            if current_price <= product['target_price']:
                print(f"🎉 {product['name']} 降价了！当前价格：¥{current_price}")
                self.send_notification(product, current_price)
            
            return current_price
            
        except Exception as e:
            print(f"检查失败：{e}")
            return None
    
    def record_price(self, name, price):
        """记录历史价格"""
        record = {
            'date': datetime.now().strftime('%Y-%m-%d %H:%M:%S'),
            'price': price
        }
        
        filename = f"{name}_price_history.json"
        try:
            with open(filename, 'r', encoding='utf-8') as f:
                history = json.load(f)
        except:
            history = []
        
        history.append(record)
        
        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(history, f, ensure_ascii=False, indent=2)
    
    def send_notification(self, product, price):
        """发送通知（这里用打印代替，可改成邮件/微信）"""
        print(f"""
        ╔════════════════════════════════════╗
        ║     🎉 降价提醒！                  ║
        ╠════════════════════════════════════╣
        ║ 商品：{product['name']}
        ║ 当前价：¥{price}
        ║ 目标价：¥{product['target_price']}
        ║ 链接：{product['url']}
        ╚════════════════════════════════════╝
        """)
    
    def run(self):
        """运行监控"""
        print("开始监控商品价格...")
        for product in self.products:
            print(f"检查：{product['name']}")
            self.check_price(product)
            time.sleep(2)  # 避免请求过快

if __name__ == "__main__":
    monitor = PriceMonitor()
    
    # 添加要监控的商品
    # monitor.add_product("iPhone 15", "https://...", 5999)
    # monitor.add_product("MacBook Pro", "https://...", 12999)
    
    # 运行监控
    monitor.run()
```

## 第六部分：常见问题

### Q1：爬虫违法吗？

**A：** 爬取公开数据本身不违法，但要注意：
- 不要爬取个人隐私
- 不要绕过付费墙
- 不要高频请求搞垮服务器
- 遵守网站 robots.txt

### Q2：被封 IP 怎么办？

**A：** 
1. 降低请求频率（加 time.sleep）
2. 使用代理 IP
3. 更换 User-Agent
4. 使用 Selenium 模拟真人

### Q3：动态网页怎么爬？

**A：** 用 Selenium 或 Playwright，它们可以执行 JavaScript。

### Q4：数据怎么存储？

**A：** 
- 小数据：CSV、JSON
- 中数据：SQLite
- 大数据：MySQL、MongoDB

### Q5：能商业化吗？

**A：** 谨慎！建议：
- 只爬取公开数据
- 控制使用范围
- 咨询法律意见
