## 什么是提示词（Prompt）？

提示词就是你给 AI 的指令。

**好的提示词 = 好的回答**
**差的提示词 = 差的回答**

这就是为什么有些人用 AI 效率翻倍，有些人却觉得 AI 很笨。

---

## 提示词的 5 个核心要素

### 1. 角色设定

告诉 AI 它是谁：

```
❌ 差：帮我写个邮件

✅ 好：你是一位有 10 年经验的销售经理，帮我写一封客户跟进邮件
```

### 2. 具体任务

明确告诉 AI 要做什么：

```
❌ 差：分析一下数据

✅ 好：分析这份销售数据，找出月度趋势、Top3 产品、异常波动
```

### 3. 背景信息

提供必要的上下文：

```
❌ 差：写个方案

✅ 好：我们是做 SaaS 的，目标客户是中小企业，预算 10 万，写一份营销方案
```

### 4. 输出格式

指定想要的格式：

```
❌ 差：整理一下

✅ 好：整理成表格，包含姓名、邮箱、公司、备注四列
```

### 5. 约束条件

设定限制和要求：

```
❌ 差：随便写写

✅ 好：500 字以内，语气正式，不要使用专业术语
```

## 100+ 个高质量提示词模板

### 工作场景

#### 邮件写作

```
你是一位专业的商务秘书，帮我写一封【邮件类型】邮件：
- 收件人：【对方身份】
- 目的：【要达成什么】
- 背景：【相关情况】
- 语气：【正式/  casual/ 诚恳】
- 关键点：【必须包含的内容】
```

#### 会议纪要

```
请帮我整理会议纪要，包含以下内容：
1. 会议基本信息（时间、地点、参会人）
2. 讨论要点（分条列出）
3. 决策事项（明确结论）
4. 待办任务（责任人 + 截止时间）

会议记录：【粘贴记录】
```

#### 周报月报

```
请帮我写一份【周/月】报，结构如下：
1. 本周/月工作内容（3-5 项重点）
2. 工作成果（用数据说话）
3. 遇到的问题及解决方案
4. 下周/月工作计划

具体内容：【粘贴工作内容】
```

### 学习场景

#### 概念解释

```
我想学习【概念/主题】，请用通俗易懂的方式解释：
1. 基本定义（一句话）
2. 核心原理（类比说明）
3. 实际应用场景（3 个例子）
4. 常见误区（2-3 个）

适合【零基础/有基础】的人理解
```

#### 学习计划

```
我想在【时间】内学会【技能】，请帮我制定学习计划：
- 当前水平：【零基础/入门/进阶】
- 每天可投入：【X 小时】
- 学习目标：【能做什么】

请提供：
1. 学习路径（分阶段）
2. 每个阶段的学习内容
3. 推荐资源（书籍/课程/网站）
4. 实践项目建议
```

### 编程场景

#### 代码生成

```
你是一位资深【语言】开发者，请帮我写一个【功能】：
- 输入：【输入数据/格式】
- 输出：【期望结果】
- 要求：【性能/安全/可读性】
- 需要注释和示例

请提供完整代码和使用说明
```

#### 代码调试

```
这段代码报错了，请帮我找出问题：

错误信息：【粘贴错误】
代码：【粘贴代码】
预期功能：【应该做什么】

请：
1. 指出问题所在
2. 解释原因
3. 提供修复方案
```

### 创作场景

#### 文章写作

```
请帮我写一篇【文章类型】：
- 主题：【核心主题】
- 受众：【目标读者】
- 字数：【大约多少字】
- 风格：【正式/ 轻松/ 专业/ 幽默】
- 要点：【必须包含的要点】
- 结构：【希望的段落结构】
```

#### 头脑风暴

```
我要做【项目/产品】，请帮我头脑风暴：
- 目标用户：【谁】
- 核心功能：【什么】
- 竞争优势：【为什么选我】

请提供：
1. 20 个创意点子
2. 每个点子的可行性分析
3. 推荐的前 3 个方向
```

## 高级提示词技巧

### 技巧 1：分步思考

```
请逐步思考这个问题：
1. 先分析问题本质
2. 列出可能的方案
3. 评估每个方案的优劣
4. 给出最终建议

问题是：【你的问题】
```

### 技巧 2：角色扮演

```
你是一位【角色】，有【X 年】经验，擅长【领域】。

现在请【任务描述】。

请用【角色】的专业视角和语言风格回答。
```

### 技巧 3：对比分析

```
请对比【A】和【B】：
1. 各自的优势
2. 各自的劣势
3. 适用场景
4. 我的情况应该选哪个

我的情况：【描述你的情况】
```

### 技巧 4：让 AI 自我质疑

```
请给出你的方案，然后：
1. 指出这个方案可能的 3 个问题
2. 给出改进建议
3. 提供一个备选方案
```

### 技巧 5：提供示例

```
请按照以下示例的格式和风格，帮我写【内容】：

示例：【粘贴示例】

我的内容：【描述你的内容】
```

## 提示词优化前后对比

### 案例 1：写邮件

**优化前：**
```
帮我写个邮件
```

**优化后：**
```
你是一位专业的商务秘书，帮我写一封客户跟进邮件：
- 客户上周咨询了产品价格
- 说需要考虑一下
- 现在要礼貌地询问进展
- 语气专业但不生硬
- 100 字以内
```

**效果对比：**
- 优化前：AI 不知道写给谁、什么目的
- 优化后：直接可用，只需微调

---

### 案例 2：写代码

**优化前：**
```
写个 Python 脚本
```

**优化后：**
```
你是一位资深 Python 开发者，请帮我写一个文件批量重命名脚本：
- 输入：文件夹路径和前缀
- 功能：按序号重命名所有文件
- 输出：重命名后的文件列表
- 要求：有错误处理，有使用示例
- 需要详细注释
```

**效果对比：**
- 优化前：AI 不知道要什么功能
- 优化后：完整可用的脚本

## 常见错误

### ❌ 错误 1：问题太模糊

```
帮我写点什么
```

**解决：** 明确告诉 AI 你要什么

---

### ❌ 错误 2：一次性问太多

```
帮我写个邮件，再分析一下数据，还要写个方案...
```

**解决：** 拆分问题，逐个击破

---

### ❌ 错误 3：不验证输出

```
AI 说什么信什么
```

**解决：** 重要信息要核实

---

### ❌ 错误 4：期望完美

```
期望 AI 一次给出完美答案
```

**解决：** 迭代优化，多轮对话
