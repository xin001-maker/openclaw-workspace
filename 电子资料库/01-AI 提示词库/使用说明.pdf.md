## 💡 使用方法

### 1. 直接复制使用

每条提示词都是完整模板，直接复制后：
1. 替换 `[填写...]` 部分为你的具体内容
2. 发送给 AI（ChatGPT/Claude/文心一言等）
3. 获得高质量回复

**示例：**
```
【爆款标题生成器】
请为以下主题生成 10 个爆款标题：
主题：[AI 工具如何提高工作效率]

要求：
- 使用数字、疑问、对比等技巧
- 制造好奇心和紧迫感
- 符合新媒体传播规律
- 每个标题不超过 25 字
```

### 2. 组合使用

可以组合多条提示词使用，例如：
1. 先用【文章大纲生成器】生成大纲
2. 再用【万能文章生成器】撰写正文
3. 最后用【文章润色优化】优化

### 3. 调整优化

根据实际效果调整：
- 增加更具体的要求
- 调整语气风格
- 添加示例参考

## 🎯 使用技巧

### 1. 结构化输入

**好的输入：**
```
请帮我写一篇文章：
- 主题：时间管理
- 目标读者：职场新人
- 文章类型：干货
- 字数：2000 字
- 语气：专业但易懂
- 核心要点：1.优先级 2.番茄工作法 3.工具推荐
```

**差的输入：**
```
帮我写篇时间管理的文章
```

### 2. 迭代优化

第一次输出不满意时：
- 说明具体问题（"太笼统了，请更具体"）
- 添加要求（"请加入实际案例"）
- 调整风格（"请更幽默一些"）

### 3. 角色设定

给 AI 设定专业角色：
```
你是一位有 10 年经验的市场营销专家...
你是一位资深程序员...
你是一位专业心理咨询师...
```

### 4. 提供示例

给出示例让 AI 学习：
```
请按照以下风格写作：
[粘贴示例文章]
```
