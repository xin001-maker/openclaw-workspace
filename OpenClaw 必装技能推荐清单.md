## ⚠️ 安全提醒

### 安装前检查清单

1. ✅ **查看源代码**
   - 访问技能的 GitHub 仓库
   - 检查 SKILL.md 内容
   - 确认无恶意代码

2. ✅ **检查作者**
   - 查看作者历史技能
   - 确认是可信开发者
   - 查看社区评价

3. ✅ **权限审查**
   - 技能需要的权限
   - 是否过度授权
   - 数据访问范围

4. ✅ **VirusTotal 扫描**
   - 在 ClawHub 查看 VirusTotal 报告
   - 确认无安全标记

### 高风险技能类型

❌ **避免安装**:
- 要求系统 root 权限
- 访问敏感文件（密码、密钥）
- 网络请求到不明域名
- 加密/混淆代码
- 无源代码或描述过简
